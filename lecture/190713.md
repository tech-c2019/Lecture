## Lecture

### 文字チェック
```
# (1)/を使う書き方
value = "hello"
if /hello/ =~ value 
  puts "match"
end
```

```
value = "hello"
reg = Regexp.new("hello")
if reg =~ value
  puts "match"
end
```

```
value = "hello"
if %r!hello! =~ value
  puts "match"
end
```

### 郵便番号チェック
```
value = "1910055"
if %r!^[0-9][0-9][0-9]-[0-9][0-9][0-9][0-9]! =~ value
  puts "match"
end
```

```
value = "1910055"
if %r!^\d{3}-\d{4}$! =~ value
  puts "match"
end
```
